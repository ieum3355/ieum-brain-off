# 데이터 무결성 및 실시간성 규칙 (Data Integrity Rules)

본 프로젝트의 모든 시세 데이터는 다음 무결성 규칙을 엄격히 준수해야 합니다.

1.  **5초 타임스탬프 원칙**:
    - 모든 종목 데이터는 `timestamp` 필드를 포함해야 합니다.
    - 현재 시스템 시간과 데이터의 `timestamp` 차이가 **5초 이상**일 경우, 해당 데이터는 '과거 데이터'로 간주하고 즉시 폐기합니다.
    - 폐기 후에는 반드시 실시간 브릿지(WebSocket)로부터 새로운 데이터를 재조회하거나 대기해야 합니다.

2.  **캐시 비활성화**:
    - 시세 조회 도구 호출 시 `cache_results=False`를 명시하여, 이전 대화 상태의 낡은 가격 정보를 재재사용하지 않습니다.

3.  **수치 기반 기계적 판단**:
    - "오른 것 같다", "분위기가 좋다"와 같은 주관적 표현을 배제합니다.
    - 오직 실시간 체결강도(%), 전일 대비율(%), 시가 대비 상승률(%) 등의 수치 데이터만으로 '맥점'을 판정합니다.

4.  **출처 명시**:
    - 모든 추천 및 분석 결과에는 해당 데이터를 가져온 소스(KIS WebSocket)와 정확한 시각을 명시합니다.
